# Colors, Icons and *Fonts*

### Colors:
`~r~` = Red
`~b~` = Blue
`~o~` = Orange
`~c~` = Grey
`~f~` = Light Blue
`~l~` or ~u~ or ~v~ = Black
`~g~` = Green
`~y~` = Yellow
`~p~` = Purple
`~m~` = Dark Grey
`~s~` or ~t~ = White
`~d~` = Dark Blue
`~q~` = Pink

### Special/Icons:
`÷` = Rockstar Icon
`∑`/`~ex_r~` = Rockstar Icon #2
`¦` = Rockstar Verified Icon
`Ω` = Lock icon
`~ws~` = Wanted Star

### Misc:
`~bold~` or `~h~` = Bold
`~italic~` or `<i>` = Italic
`~n~` = Skip Line
